# Installing RAWBY on Android (APK)

Follow these steps to install RAWBY directly on your Android phone — no Play Store needed.

---

## What you need

- Android phone (Android 8.0 or newer)
- The RAWBY APK file (sent to you by the RAWBY team)

---

## Step 1 — Allow installs from unknown sources

Android blocks APK installs by default. You need to enable this once.

1. Go to **Settings**
2. Search for **"Install unknown apps"** or navigate to **Apps → Special app access → Install unknown apps**
3. Find your file manager or browser (whichever you'll use to open the APK)
4. Toggle **"Allow from this source"** ON

> On Samsung devices this is under **Settings → Biometrics and security → Install unknown apps**

---

## Step 2 — Download the APK

1. Open the download link sent to you (or move the APK file to your phone)
2. If prompted, tap **Download anyway** — the file is safe

---

## Step 3 — Install the APK

1. Open your **Files** app (or **Downloads** folder)
2. Tap the **RAWBY.apk** file
3. Tap **Install**
4. Wait for installation to complete
5. Tap **Open**

---

## Step 4 — Log in or create an account

- Open RAWBY
- Tap **Create account** if you're new, or **Log in** if you already have one
- Start your first weekly challenge!

---

## Troubleshooting

| Problem | Fix |
|---------|-----|
| "App not installed" | Make sure you enabled unknown sources for the right app |
| "Parse error" | The APK may be corrupted — re-download it |
| App won't open | Restart your phone and try again |
| Can't find the APK | Check your **Downloads** folder in the Files app |

---

## Updates

When a new version of RAWBY is released, you'll receive a new APK link. Simply install it over the existing app — your data is saved.
